# password-paper

パスワード用紙生成スクリプト。A4用紙にランダムな文字をグリッド状に配置したPDFを生成します。

## 用途

- 複雑なパスワードを自分で考えるのが難しい
- パスワードを手書きするときにミスしやすい

このような課題を解決するため、ランダムな文字が印刷された用紙を生成し、必要な部分をハサミで切り取ってパスワードノートに貼り付けて使用します。

## 特徴

- 各文字が四角で囲まれており、ハサミで切りやすい
- 誤読しやすい文字（l, 1, I, O, 0, o）を除外
- 大文字・小文字・数字をバランスよく配置（40:40:20）
- 暗号学的に安全な乱数生成器（`secrets`モジュール）を使用

## 必要条件

- Python 3.10以上
- [uv](https://docs.astral.sh/uv/)

## 使い方

```bash
uv run main.py
```

`password_sheet.pdf` が生成されます。

## 出力例

- グリッドサイズ: 約20列 x 35行
- ヘッダー行に列番号を表示
- フッターに生成日を表示

## Vercel へのデプロイ

このリポジトリはVercelにデプロイしてWebサービスとして使用できます。

### デプロイ手順

1. [Vercel](https://vercel.com) にログイン
2. 「Add New Project」からこのリポジトリをインポート
3. そのままデプロイ

### 使い方

デプロイ後、以下のURLにアクセスするとPDFがダウンロードされます：

```
https://your-project.vercel.app/api/generate
```
